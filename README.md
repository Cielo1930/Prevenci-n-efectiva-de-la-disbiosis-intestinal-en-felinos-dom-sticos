        <h2>1. Seguimiento Clínico Regular</h2>
        <ul>
            <li>Consultas veterinarias cada 3-6 meses.</li>
            <li>Evaluación de peso, digestión, apetito y comportamiento.</li>
            <li>Registro y análisis de signos digestivos persistentes.</li>
        </ul>

        <h2>2. Educación al Tutor</h2>
        <ul>
            <li>Implementación de ambientes enriquecidos.</li>
            <li>Rutinas estables para reducir estrés.</li>
            <li>Identificación y manejo de ansiedad felina.</li>
        </ul>

        <h2>3. Control Nutricional</h2>
        <ul>
            <li>Dieta gastrointestinal de alta calidad.</li>
            <li>Evitar comida casera no balanceada.</li>
            <li>Uso adecuado de prebióticos y probióticos.</li>
        </ul>

        <h2>4. Uso Responsable de Medicamentos</h2>
        <ul>
            <li>Evitar tratamientos sin diagnóstico.</li>
            <li>Uso de antibiogramas antes de prescribir.</li>
            <li>Prevención de resistencia antimicrobiana.</li>
        </ul>
